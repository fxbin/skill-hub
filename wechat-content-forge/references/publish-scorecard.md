# Publish Scorecard

发布门禁除了 PASS / WARN / FAIL，还要尽量给出可复查的评分卡。

## 评分维度

每项按 `0-2` 分打分：

- `0`: 明显失败，必须回退
- `1`: 勉强成立，但仍需修改
- `2`: 达到当前阶段可发标准

## 评分项

### 像不像我
- 第一人称真实感
- 经验到判断的自然过渡
- 语气克制度

### 能不能发
- 开头抓力
- 结构推进
- 交付清晰度
- 事实稳定度

### 发了值不值
- 新意
- 可带走价值
- 公共表达完成度

## 判定规则

- 任一关键项为 `0`：`FAIL`
- 总体存在多个 `1`：`WARN`
- 关键项都不为 `0`，且多数为 `2`：`PASS`

关键项：
- 开头抓力
- 结构推进
- 交付清晰度
- 事实稳定度

## 输出模板

- `评分卡`
- `关键失分项`
- `致命问题`
- `建议回退环节`
- `最终结论`
