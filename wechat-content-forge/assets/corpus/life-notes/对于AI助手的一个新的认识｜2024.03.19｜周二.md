ollama 一些常用命令记录：

```shell
# 查询版本
ollama --version 

# 下载 llama2-Chinese 7B 模型
ollama pull llama2-chinese

# 下载 llama2-Chinese 13B 模型
ollama pull llama2-chinese:13b

# 使用， 以llama2-chinese 模型为例
ollama run llama2-chinese:13b "天空为什么是蓝色的？"

# 服务器模式
ollama serve

```
