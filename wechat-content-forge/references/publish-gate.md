# Publish Gate

这是一道比一般审核更严格的发布门禁。默认立场不是“差不多能发”，而是“没有足够证据就先不发”。

## 三层门禁

### 1. 像不像我

必须确认：

- 是否保留第一人称真实感
- 是否保留从经验到判断的自然推进
- 是否避免模板腔、营销腔、过度光滑的 AI 腔

### 2. 能不能发

必须确认：

- 开头是否有抓力
- 结构是否在推进，而不是原地兜圈
- 是否真有交付，而不是只有感受或资料堆积
- 事实、引用、产品信息是否稳定

### 3. 发了值不值

必须确认：

- 读者能带走什么
- 这篇相比旧文的新意是什么
- 这篇是否已经完成公共表达，而不是私人记录外泄

## 门禁输出格式

- `像不像我`: PASS / WARN / FAIL
- `能不能发`: PASS / WARN / FAIL
- `发了值不值`: PASS / WARN / FAIL
- `最强处`
- `致命问题`
- `建议回退环节`
- `最终结论`: 可发 / 修改后再审 / 不建议发

## 回退规则

- 如果文风失真，回到 `文风确认`
- 如果结构不走，回到 `选题确认`
- 如果事实不稳，回到 `资料检索 / 研究回路`
- 如果交付不清，回到 `主结论与结构骨架`
